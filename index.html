HTML

  • The head element contains information about the webpage.
  • The body element represents the visable contents shown to the user.
  • H1 to H6 are heading sizes, the smaller the number the bigger the font.
  • ul creates an unordered list of items.
  • ol creates ordered list with numbers next to the bullet points
  • li creates bullet points to a list.

CSS

  • CSS stands for Cascading style sheets
  • there are 3 types of CSS, inline, internal, and external
  • Inline is where you add css directily on the line you want modified
  • Internal is where you add Css in the head of the Html to affect the whole page
  • External is where you but you Css code in a different file and link it in the Html doc

Git

  • git add -A to add all new files into the current branch
  • git status to see your current branch and if anything hasnt been commited to it
  • git checkout main to switch to main branch
  • git checkout "branch name" to switch to that branch
  • git checkout -b to make a new branch
  • git commmit -m "commit message" commmits your progress for a PR!

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Js add function to your webpage
  • Use for loops to loop through a set of items
